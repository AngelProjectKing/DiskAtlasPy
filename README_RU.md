# DiskAtlasPy v3 — фиксы UX/анимаций/статистики

Основные изменения по твоим жалобам:
1) **Больше не “прыгает” прогресс**: убран 32-бит overflow (у тебя уходило в минус после ~2GB).
   Прогресс теперь по **% от размера диска** (оценка через psutil.disk_usage).
2) **Сплиттеры больше не “прячутся”**: видимый handle, шире, не коллапсится, есть кнопка “Сбросить”.
3) **Статистика стала понятной**:
   - Использование дисков с прогресс-барами
   - Топ расширений с % и прогресс-барами
   - Топ больших файлов
   - Короткая сводка “что занимает место”
4) **Защита**: понятные шаги + авто-выбор выходного имени:
   - Шифрование -> добавляет `.datlas`
   - Расшифровка -> убирает `.datlas`
   - Авто-детект формата (DiskAtlas) и подсказки
5) Больше анимаций:
   - Overlay с fade-in/fade-out
   - Treemap плавно появляется
   - Плавное обновление % прогресса

## Запуск
```bat
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
python main.py
```
